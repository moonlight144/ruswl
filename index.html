Hiddify

Инструкция

1️⃣ Установка: Скачайте приложение Hiddify. Откройте и разрешите доступ к уведомлениям и VPN-профилю.
2️⃣ Первичная настройка: После первого запуска зайдите в:
• Настройки → Общие → Ограничение памяти → Включить
• Настройки → Общие → Использовать Xray-core → Включить
• Настройки → Маршрутизация → Блокировать рекламу → Включить
3️⃣ Конфиг: Выберите нужный список внизу и нажмите кнопку "ИМПОРТ". Приложение само добавит профиль.
4️⃣ Запуск: Нажмите на круглую кнопку "Подключиться". После появления задержки (мс) выберите сервер с пометкой lowest для лучшей скорости.
